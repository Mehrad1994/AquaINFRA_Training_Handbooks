## Accessing the Package

To access the **Pan-European Data to Knowledge Package**, follow these steps on the **AquaINFRA Interaction Platform**:

1.  Go to the [AquaINFRA Interaction Platform](https://aquainfra.dev.52north.org).
2.  In the search bar, type **"Data to Knowledge Package"** and click Search.
3.  Select the **Pan-European package**: *"Data to Knowledge Package for Specleaner: A toolkit for ensemble outlier detection and removal"*.

## Package Components

The package page provides a clear overview of the workflow and resources:

-   **Virtual Research Environment (Workflow)**:
    -   Clicking on the workflow provides detailed documentation on platforms such as **Galaxy** and **Zenodo**.
    -   Each step is described for easy exploration.

-   **Virtual Lab**:
    -   Redirects to a **Binder** environment where you can interact with R and Python code and explore the Docker image in your browser.

-   **Web API**:
    -   Reviews access to PyGeoAPI services linked to the workflow.
    -   Pan-European processes are listed under **"BOKU"**.

-   **Reproducible Basis**:
    -   Contains the complete toolbox, including scripts and tools hosted on GitHub and Zenodo.

-   **Datasets**:
    -   Provides detailed info on input datasets required for the workflow.
    -   Includes direct download links (e.g., for the **GeoJSON** file defining the area of interest).

This structure ensures all components are **FAIR** (Findable, Accessible, Interoperable, Reusable).
