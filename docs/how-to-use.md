# How to Use This Handbook

This site hosts a collection of training materials and use case documentation for AquaINFRA.

## Navigation
Use the sidebar on the left to navigate between different sections:
- **Guide**: General information about the handbooks.
- **Resources**: Links to external AquaINFRA resources and OERs.
- **Use Cases**: Specific training modules grouped by region.

## Structure
Each training module typically includes:
1. **Introduction**: Scientific background.
2. **Workflow**: Step-by-step instructions.
3. **Results**: Expected outputs and interpretation.

## Feedback
If you encounter any issues or have questions, please contact the AquaINFRA support team.
