# Running the Workflow

This chapter guides you through importing and running the Pan-European workflow in the **AquaGalaxy** VRE.

## 1. Import Workflow to AquaGalaxy
1.  Make sure you are logged in to [Aqua Galaxy](https://aqua.usegalaxy.eu).
2.  On the **Data to Knowledge Package** page (Interaction Platform), click on the **Workflow** section.
3.  Select **Galaxy**. You will be redirected to Aqua Galaxy.
4.  Click **"Import workflow into Galaxy"** (or "Start using this workflow").
5.  Confirm the import. The workflow should now appear in your Galaxy interface.

## 2. Get Input Data
1.  Return to the **Data to Knowledge Package** page > **Datasets** > **Zenodo**.
2.  Locate the **GeoJSON file** (defines the area of interest).
3.  Right-click the download link and **Copy link address**.
4.  Return to **Aqua Galaxy**.
5.  Open the **Upload Data** tool:
    *   Paste the copied URL.
    *   Click **Start**.
6.  Once the box turns **green**, the data is imported.
7.  *(Optional)* Rename the dataset to "Area of Interest" by clicking the pencil icon.

## 3. Run the Workflow
1.  Go to the **Workflow** section in Galaxy.
2.  Choose the imported **Pan-European Workflow**.
3.  Click the **Run** button (play icon).
4.  On the run page:
    *   Select your input dataset (the GeoJSON file) for the corresponding input field.
5.  Click **Run Workflow**.

It will take some time for all steps to complete (turn green).
